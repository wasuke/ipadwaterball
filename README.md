# iPad Water Ball Demo

黒背景（有機EL想定）上に、カメラで取得した手の動きに応じて移動する青い水球を表示するデモです。  
人差し指と親指で触れると、水球は柔らかく凹み、内部がゆらぎながら反発して元の形に戻ります。

---

## 🔵 デモページ

👉 https://wasuke.github.io/ipadwaterball/

---

## 概要

本デモは以下の要素で構成されています。

- Webカメラ入力（内蔵・USB外付け対応）
- MediaPipe Hands による手指追跡
- 2D Canvas によるリアルタイム描画
- 物理モデル（ばね・減衰）による移動挙動
- 指接触による移動
- 球内部の流体的揺らぎ表現

背景は完全な黒（#000）で固定しており、有機ELディスプレイでの黒の沈み込みを前提に設計しています。

---

## 操作方法

1. ページを開く
2. カメラ一覧を更新し、使用するカメラを選択
3. 「このカメラで開始」を押す
4. 人差し指で水球に触れる

### 動作

- 近づくと球が凹む
- 強く突くと弾かれるように移動
- 離すとばねモデルにより中心へ復帰
- 内部の濃淡が時間変化し流体感を演出

---

## 技術構成

### 手検出
- MediaPipe Hands（CDN読み込み）
- index finger tip（ランドマーク8）を使用

### 描画
- Canvas 2D
- RadialGradient による体積表現
- screen ブレンドによる内部揺らぎ
- クリップによる内部限定描画

---

## 対応環境

- Safari（iPadOS / macOS）
- Chrome
- USB外付けカメラ対応
- WebGL不要（Canvas 2Dのみ）

※ カメラのラベルを表示するため、最初にカメラ権限の許可が必要です。
